Prework Study Guide

HTML

  • The head element contains information about the webpage
  • The body element represents the visable content shown to the user
  • HTML uses a system of tags and attributes to define the structure of content on a webpage
  • "It's a website"

CSS

  • CSS is used to style the webpage
  • It can be used to change the color, size, and position of elements on a webpage
  • It can be used to create animations and transitions
  • A margin indicates how much space we want around the outside of an element
  • A padding indicates how much space we want around the content inside an element
  • "It's a website, but make it pretty"

GIT

  • GIT is a version control system used to track changes in code
  • It is used to collaborate with other developers
  • It is used to deploy code to a server
  • "It's a website, but make it pretty, and make it work"
  • git status: checks what branch you are currently on
  • git checkout -b branch-name: creates a new branch and switches to it

JavaScript

  • JavaScript is a programming language used to create interactive elements on a webpage
  • It can be used to create games, forms, and other interactive elements
  • It can be used to create animations and transitions
  • "It's a website, but make it pretty, make it work, and make it interactive"
